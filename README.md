# Old World Builder

The "Old World Builder" is an army builder for the [Games Workshop](https://www.games-workshop.com) tabletop game Warhammer: The Old World.

You can visit the site here: [old-world-builder.com](https://old-world-builder.com/)

## What This Is

This is a fork of the popular Old World Builder army builder for Warhammer: The Old World. It has been modified to automatically import the German Comp json ruleset and display modified stats and rules for German Comp, which the original OWB cannnot do, but is otherwise the same.
